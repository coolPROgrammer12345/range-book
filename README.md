# Range Book

A browser-based golf practice tracker built as a single-file HTML app.

## Features
- Putting, Chipping, and Driver practice sections with 8 drills each,
  each with Beginner / Intermediate / Advanced goal tiers
- Session logging (attempts, makes, notes) saved to persistent storage
- History tab with stats (total sessions, sessions this week, day streak)
- Nearby tab: finds golf courses and driving ranges within 25 miles
  using OpenStreetMap/Overpass data, with an embedded map and directions links

## Usage
Open `range-book.html` in a browser.
